# Changelog

## v0.9.3-rc2

- 修复重复同步在线配置时生产状态反复提示和弹窗的问题。
- 螺钉枪按方向、状态、触发快照处理 OK、NG 和反转事件，业务处理后再清除地址53。
- 降级模式暂停螺钉枪防错轮询，并主动写地址4=1放开螺钉枪。
- 所有桌面端管理员密码验证和设置入口统一使用触摸式数字键盘。
- 修复 cancelcode 取消当前工位不同工序条码失败，并增加匹配失败调试信息。
- 客户端生产接口继续使用 station session 校验，不依赖 Web 管理登录。

## v0.9.3-rc1

- 增加主条码本工位防重复和非主条码跨位置防重复。
- 增加 cancelcode 取消模式及降级模式日志。
- 增加螺钉枪 NG=3/4、反转锁枪和 PLC 首工序支持。
